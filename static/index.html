Ronan's Rover Controller

Use the command line interface below to program the rover

Commands
  • f [#] | forward [#] -- move forward for # sec
  • b [#] | backward [#] -- move backward for # sec
  • r [#] | spinright [#] -- spin right for # sec
  • l [#] | or spinleft [#] -- spin left for # sec
  • rf [#] -- move right & forward for # sec
  • rb [#] -- move right & backward for # sec
  • lf [#] -- move left & forward for # sec
  • lb [#] -- move left & back for # sec
  • Time intervals in seconds
  • Omit the time will default to 1 second
  • Maximum of 5 seconds allowed
